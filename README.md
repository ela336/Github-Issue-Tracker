1.What is the difference between var, let, and const?
Ans:

var:
-Old way.
-Function scoped.
-Can be re-declared and updated.

let:
-Block scoped.
-Can be updated, but can not be re-declared.

const:
-Block scoped
-Cannot be updated or re-declared.
